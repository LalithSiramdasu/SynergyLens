Project Note

NCI ALMANAC ComboScore Deployment

SynergyLens is a Flask deployment app for final Step 6 models. It validates NSC inputs, builds the exact 526-feature model vector, selects the best model per cell line, predicts both drug orders, and averages the final ComboScore.

Target ComboScore

Positive values suggest stronger-than-expected inhibition and synergy. Negative values suggest antagonism.

Model Automatic Selection

The backend uses the final Step 6 registry. Users never choose RandomForest, XGBoost, CatBoost, or LightGBM manually.

Safety Screening Only

Predictions and XAI explanations are machine-learning screening estimates, not calibrated probabilities, biological proof, or clinical advice. Promising synergy predictions should be validated experimentally.
